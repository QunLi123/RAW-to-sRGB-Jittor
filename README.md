# RAW-to-sRGB-Jittor
RAW-to-sRGB Jittor 复现

## 环境配置
Ubuntu 22.04
Python 3.12.12
Jittor 1.3.9.14
CUDA 12.1

## 数据集
请使用网盘链接下载ZRR数据集 https://pan.baidu.com/s/1NpJacIOowSsfsIgJZrmjKg?pwd=8sq1&_at_=1771844223136#list/path=%2F

## ckpt文件夹
请使用网盘链接下载并放至项目根目录中 https://pan.baidu.com/s/1Sdd7yYFATLM6bMyOI97iJw 
提取码：uis3

## 训练方式
请下载数据集，并移动至train_zrr.sh路径，执行 bash train_zrr.sh 即可
